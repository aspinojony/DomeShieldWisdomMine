## 1. 开始之前

### 1.1 这是什么软件

**穹盾智矿** 是一套矿山安全监测演示系统，装好后会在你的电脑上打开一个**网页大屏**，
展示矿山数据、AI 风险预警、无人机视觉识别等内容。

**你只需要会用浏览器，剩下软件自己搞定。**

### 1.2 你需要准备什么

| 项目 | 要求 |
|---|---|
| 操作系统 | Windows 10 或 Windows 11（64 位） |
| 硬盘空间 | 至少 5 GB 空闲 |
| 内存 | 至少 8 GB |
| 网络 | 安装时**不需要联网**；安装好后也**不需要联网**就能用 |
| 安装包 | 一个名叫 `穹盾智矿-Setup-1.0.0.exe` 的文件（对方会发给你） |

> Windows 7 / Windows 8 / Mac / 手机 **不能装**，请不要尝试。

### 1.3 重要提醒

- 整个软件包大约 **1.5 GB**，下载比较慢是正常的
- **安装过程中 Windows 会弹两次"警告"窗口**，这是因为这是一个新软件，
  没有买微软的"通行证"，**不是病毒**，按本手册指引点击即可
- 装完会在桌面留一个 **蓝色盾牌图标**

## 2. 第一次安装（约 5 分钟）

### 步骤 1：双击安装包

找到对方发给你的 `穹盾智矿-Setup-1.0.0.exe` 文件，**鼠标左键双击**。

### 步骤 2：处理"Windows 已保护你的电脑"蓝色窗口

双击后，屏幕中央会弹出一个**蓝色窗口**，标题是 **"Windows 已保护你的电脑"**。

> **这是 Windows 的默认提示，不是病毒，按下面操作即可。**

操作：

1. 看到蓝色窗口后，**先不要点右下角的"不运行"**
2. 在窗口中间找到一行小字：**"更多信息"** —— **点击它**
3. 这时窗口会变长，多出一个 **"仍要运行"** 按钮
4. **点击"仍要运行"**

> 如果你已经不小心点了"不运行"——没关系，重新双击 `穹盾智矿-Setup-1.0.0.exe` 再来一次。

### 步骤 3：处理"用户账户控制"黄色窗口

接下来 Windows 会再弹一个**黄色窗口**，问"是否允许此应用对你的设备进行更改？"

**点"是"**。

### 步骤 4：跟着安装向导一路点"下一步"

接下来会进入正式的**安装向导**（有中文界面），按下面顺序操作：

| 屏幕显示 | 你要做 |
|---|---|
| **选择语言** | 选"简体中文"，点"确定" |
| **欢迎页面** | 点"下一步" |
| **安装位置** | 不用改，直接点"下一步" |
| **附加任务** | **保持"在桌面创建快捷方式"勾选**（默认勾上的），点"下一步" |
| **准备安装** | 点"安装" |
| **正在安装** | **等 1-3 分钟**，进度条会走完 |
| **完成** | **保持"立即启动 穹盾智矿"勾选**，点"完成" |

## 3. 第一次启动（约 30 秒）

安装完成后，软件会**自动启动**。如果你不小心关掉了，**双击桌面上的"穹盾智矿"图标**也可以启动。

### 3.1 处理"Windows 防火墙"安全警告

启动后，屏幕会弹出一个**白色窗口**，标题是 **"Windows 安全中心警报"** 或 **"Windows Defender 防火墙已阻止此应用的某些功能"**。

**操作（很重要，不要点错）：**

1. 看到窗口里有两个勾选框：**"专用网络"** 和 **"公用网络"**
2. **只勾"专用网络"**（如果默认已经勾了，保持就行）
3. **不要勾"公用网络"**
4. 点 **"允许访问"** 按钮

> 这一步会弹**最多 5 次**（5 个内部服务各弹一次），每次都按同样操作即可。如果**一次性勾了"专用网络"再点允许**，后面通常就不会再问了。

### 3.2 等待浏览器自动打开

防火墙都允许之后，**等 20-30 秒**，浏览器（Edge / Chrome）会自动打开，显示一个**矿山大屏页面**。

如果 30 秒过去浏览器还没动，**手动**做以下任一操作：

- **方法一**：找到右下角任务栏的**蓝色盾牌小图标**（在时间旁边那一堆图标里），
  鼠标**右键**点它，选 **"打开大屏"**
- **方法二**：自己打开浏览器，地址栏输入 `http://127.0.0.1:5173` 然后回车

### 3.3 登录系统

页面会显示登录框，**第一次登录用默认账号**：

```
用户名：admin
密码：  admin123
```

进入后就能看到完整的矿山大屏了。

## 4. 日常使用

软件装好以后，**每次想用就这样**：

1. **双击桌面上的"穹盾智矿"图标**
2. 等 15-20 秒，浏览器自动开
3. 登录后正常使用

> 不再需要点"仍要运行"，也不再有防火墙弹窗——那些只在**第一次启动**时出现。

### 4.1 怎么知道软件在运行？

看屏幕右下角任务栏（时间旁边），如果有**蓝色盾牌图标**，就是软件正在运行中。

### 4.2 浏览器关了软件还在跑吗？

是的。**关浏览器 ≠ 关软件**。
浏览器只是显示界面，**真正的程序在后台**。要彻底关掉，看下一节。

## 5. 关闭程序

### 5.1 标准关法

1. 找右下角任务栏的**蓝色盾牌图标**
2. 鼠标**右键**点它
3. 在弹出菜单中选 **"退出"**
4. 几秒钟后盾牌图标消失，程序就完全关闭了

### 5.2 实在找不到盾牌图标？

任务栏右下角可能折叠了。点任务栏上**向上的小箭头 ∧**展开所有隐藏图标，盾牌就在里面。

### 5.3 关机会自动关闭吗？

会的。直接关机 / 重启电脑不会有问题，**不会损坏数据**。

## 6. 常见问题

### Q1：双击图标没反应 / 浏览器没自动开

**原因**：可能上一次没退干净，端口被占用。

**解决**：
1. 重启一下电脑
2. 重新双击桌面图标
3. 还是不行 → 拍照发给提供软件的人，附上下面 [7.2 节](#72-日志文件) 提到的日志文件

### Q2：浏览器打开了但是页面空白 / 一直转圈

**原因**：5 个后台服务还没全启动。

**解决**：
- 在浏览器里按 **F5** 刷新一下
- 或者**等 30 秒**再刷新
- 还不行 → 退出程序（[第 5 节](#5-关闭程序)）→ 重新双击图标

### Q3：登录提示密码错误

确认你输的是：
- 用户名（区分大小写）：`admin`（不是 `Admin`）
- 密码：`admin123`

如果之前自己改过密码忘了，可以**彻底重置**：
1. 退出程序
2. 按 `Win + R`，输入 `%APPDATA%\穹盾智矿` 回车
3. 删除里面的 `mining.db` 文件
4. 重新启动程序，账号回到默认 `admin/admin123`

### Q4：每次启动都要点"仍要运行"？

不会。"仍要运行"只在**安装那一次**出现，之后双击桌面图标就直接启动了。
如果你每次都看到——说明你**双击的是安装包**，不是桌面那个**蓝色盾牌图标**，看清楚。

### Q5：防火墙又弹了

如果换了网络（比如从家里 WiFi 换到公司）有可能再弹一次，按 [3.1 节](#31-处理windows-防火墙安全警告) 同样处理。

### Q6：能不能让它开机自动启动？

可以。**安装时**在"附加任务"那一步勾上"开机自启"即可。

已经装好了想加自启：
1. 按 `Win + R` 输入 `shell:startup` 回车，会打开"启动"文件夹
2. 把桌面上的"穹盾智矿"快捷方式**复制**一份到这个文件夹里
3. 下次开机就会自动启动

取消自启就把那份快捷方式删了。

### Q7：电脑变慢 / 风扇狂转

软件运行时会占用一定资源（CPU 20-40%、内存 1-2 GB），属正常。如果一直高占用：
1. 退出程序
2. 不用时彻底关掉（[第 5 节](#5-关闭程序)）

### Q8：界面显示乱码 / 显示英文

浏览器问题。在浏览器地址栏右键 → 重新加载，或者按 `Ctrl + Shift + R` 强制刷新。

## 7. 软件装在哪里 / 数据在哪里

### 7.1 程序文件

软件本体装在：
```
C:\Program Files\穹盾智矿\
```

**不要手动改这个目录的任何文件**，否则可能启动不了。

### 7.2 日志文件

软件运行的记录（出错时给开发者看）在：
```
%APPDATA%\穹盾智矿\logs\
```

> 快速打开方法：按 `Win + R` → 输入 `%APPDATA%\穹盾智矿\logs` → 回车
>
> 也可以右键托盘的**蓝色盾牌图标 → "打开日志目录"**。

里面有 5 个 `.log` 文件，**出问题时把它们全部打包发给开发者**：

- `business_api.log` —— 业务接口
- `sensor_api.log` —— 监测接口
- `ai_engine.log` —— AI 引擎
- `vision_engine.log` —— 视觉识别
- `frontend.log` —— 前端服务

### 7.3 你的数据

数据库（账号、设备、告警、上传的图片等）在：
```
%APPDATA%\穹盾智矿\mining.db
%APPDATA%\穹盾智矿\cv_results\
```

**这些文件在卸载软件后会保留**，下次重新安装数据还在。
想彻底清空就手动删 `%APPDATA%\穹盾智矿\` 整个文件夹。

## 8. 卸载

### 8.1 标准卸载

1. **先把软件关掉**（[第 5 节](#5-关闭程序)）
2. 开始菜单 → 找到 **"穹盾智矿"** 文件夹 → 点 **"卸载 穹盾智矿"**
3. 跟着向导点"是" → "下一步"
4. 完成后桌面图标会消失

也可以从"设置 → 应用 → 已安装的应用"里找到"穹盾智矿"点"卸载"。

### 8.2 卸载后还想彻底清干净

卸载不会删用户数据。如果连数据一起删：

1. 按 `Win + R` → 输入 `%APPDATA%` → 回车
2. 找到 **`穹盾智矿`** 文件夹 → 右键 → 删除
